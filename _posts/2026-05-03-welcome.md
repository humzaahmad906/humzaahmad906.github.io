This is the first post. Drop new posts as Markdown files into the `_posts/` folder using the filename pattern `YYYY-MM-DD-title.md`.

## Writing posts

Each file needs a small front-matter block at the top:

```yaml
---
title: "Your title here"
date: 2026-05-03
author: Your Name
---
```

Below the front matter, write in standard Markdown — headings, **bold**, *italic*, lists, code blocks, images, links — all work.

## What's next

Replace this post when you're ready. The page list at `/blog/` updates automatically every time you push.
